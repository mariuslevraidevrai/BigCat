-help : See the commands
-create : make a session on your device (Example : create 127.0.0.1 1234)
    -code : generate a session code to connect(Example : create 127.0.0.1 1234 code => 234.673.212.124:1234)
-join : connect to a session (Example : join 192.168.1.10 1234)
-clear : clear the console
-bye : quit bigcat
